# Cryst4l Documentation

Cryst4l is a **pure-CSS ITCSS framework** modernised for 2026.

It prioritises:
- predictable cascade (via `@layer`)
- composability over opinion
- modern CSS features instead of preprocessors

This documentation focuses on **how the system thinks** so it stays maintainable.
