Cryst4l ships six layers (in order):

## Reset
Browser normalisation only. No classes. No opinions.

## Settings
Design tokens only (CSS custom properties).

## Base
Element-level defaults (`body`, `a`, `input`, etc.).

## Objects
Layout primitives (`o-wrapper`, `o-grid`).

## Components
Semantic structure without forced styling (`c-*`).

## Utilities
Single-purpose overrides (`u-*`). Allowed to use `!important`.
